## Setup

### Automatic (preferred)

```sh
chmod +x setup.zsh
./setup.zsh
```

### Manual Setup

`ln -s .zshrc ~/.zshrc`

## Adding omz plugins

Instead of cloning into `~/.oh-my-zsh/custom/plugins` do `git submodule add git@github.com:<plugin-repo> ./omz-custom/plugins/<plugin-name>`, for themese change `plugins` to `themes`
